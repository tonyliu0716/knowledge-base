Create your Articles

This is a simple application for creating articles and categories.  Technology: scrolling-nav.js, AngularJS, and Node.js, and Express, and MongoDB.

Example: Read data from MongoDB. Categories: {"id": "..", "name": "Technology", "description": "some dummy data"}

Usage:

SET UP your MongoDB:

I have created two collections in Mongo:  categories & articles

Step 1:  Open your terminal then type:   mongod

Step 2:  Open another terminal then type:    mongo

Step 3:  Prepare your default data. Structure:    

{

"_id" : ObjectId("588235e6a8ac20228342161d"),

"name" : "Healthcare",

"description" : "Some text shows here..."

}

By using  db.categories.insert({"name": value, "description": value});

SET UP running Enviorment:

Step 1: Under the folder 'knowledgebase', run 'npm install' on your terminal.

cd knowledgebase After that, run npm install

Step 2: In order to run this application on your local server, type: npm start

START to run knowledgebase app:

Run on your bowser:  http://localhost:3000
